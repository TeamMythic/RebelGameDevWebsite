UI Generic Button Learning Scene

The UI Generic Button learning scene showcases the different ways to use this unique button. The Unity Button Component is great, but is very limited. You only can do on click events, and change the color of the button out of the box. The Rebel Game Devs Generic Button creates more options. You can change rotation, scale, position, color, lerp graph type, and so much more. You can Read up on the RGD_UI_Generic_Button here.
