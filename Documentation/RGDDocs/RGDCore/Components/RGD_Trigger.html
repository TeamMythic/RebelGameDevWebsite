RGD_Trigger

The RGD_Trigger is a utility component with the goal of making trigger calls simple. When a object that has a rigidbody enters a collider that is a trigger type then there is 3 poosible method calls.

OnTriggerEnter() {when entered the trigger}, OnTriggerExit {when exiting the trigger}, and OnTriggerStay() {while inside the trigger - called every frame}.

Requirements

- The trigger needs a collider that is set to trigger.

- The target to enter needs to have some sort of Physics component (Rigidbody or CharacterController).

RGD_Trigger Functionality

- Can take an array of either Tags or an array of Transforms for the target(s).

- Optional deletion of trigger boolean (for 1 time use).

- As well as 3 Unity Events to drag and drop in depending on the trigger type.
