RGD_2p5DController

The RGD_2p5DController component is a controller built by RGD's. This component requires a few other components.

Parameters

Functional Options:These handle movement that you can simply toggle.

Controls: Handles control key types.

Movement Parameters: These are the movement speeds. More will be added if the controller get's more movement abilities.

Jump Parameters: Handles the amount of jump forces.

Physics: Physics inteaction in the world.

Cursor:Cursor options that are applied to the player controller.

Change Parameters In Code

The RGD_2p5DController has many getter and setter methods and getter and setter variables to change parameters.
