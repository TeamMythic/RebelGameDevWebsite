RGD_UIGenericButton

The RGD_UIGenericButton is an advanced version of the unity UI button. The classic button by Unity can only change color and call a click event. The RGD_UIGenericButton can check multiple states on hover, stopHover, and clicked.

Changeable Parameters

Hovering Parameters:

Lerp Curve: Effects the way the button lerps.

Hover Time To Animate: Duration of hovering animation.

Hover Rotation Euler End: rotation when hovering.

Hover Scale End: Scale when hovering.

Hover Position End: Position when hovering.

Hover Color End: Color when hovering.

Clicked Parameters:

Button Click Action: What type of button click action to check when clicking.

Hover Time To Animate: Duration of clicked animation.

Hover Rotation Euler End: rotation when clicked.

Hover Scale End: Scale when clicked.

Hover Position End: Position when clicked.

Hover Color End: Color when clicked.

Events

The Button has 3 Unity Event calls and 3 void return Delegate calls. The Unity Events are SerializeFields so actions can be dragged and dropped in. The Delegates can be accessed in script. Please refer below on how to add a void event to be called for the button delegate. 

                                   
    public class  RGD_UI_Extra_Manager : MonoBehaviour
    {
            //Get a reference to Button component:
            [SerializeField] private  RGD_UIGenericButton button;
            
            private  void Awake()
            {
                    Add methods to corresponding delegate: 
                    button. eventWhenPressed += ClickAction;
                    button.eventWhenHovered += HoveredAction;
                    button.eventWhenStopHovering += HoveredOffAction;
            }

            private   void  ClickAction()
            {
                    This is called everytime the button is Clicked:
                    Debug.Log($"The Button Was Pressed And Called On: <color=red>{this}</color>");
            }

            private   void  HoveredAction()
            {
                    This is called everytime the button is Hovered:
                    Debug.Log($"The Button Was Hovered And Called On: <color=red>{this}</color>");
            }

            private   void  HoveredOffAction()
            {
                    This is called everytime the button is Hovered Off:
                    Debug.Log($"The Button Was Hovered off And Called On: <color=red>{this}</color>");
            }
    }
